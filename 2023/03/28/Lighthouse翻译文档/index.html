Lighthouse翻译文档

发表于 2023-03-28 本文字数： 634 阅读时长 ≈ 2 分钟

Lighthouse是Google开发用来测试网页性能的工具，来自Google Lighthouse Readme。本篇文章简单翻译了如何使用Lighthouse，并没有追求直译。

Lighthouse

Lighthouse可以分析web应用和网页，收集在开发者看来最佳的性能指标和观点。

使用Lighthouse

在Chrome中使用Lighthouse

Lighthouse直接集成到Chrome的开发者工具中，位于“Lighthouse”面板下（开发者工具-Settings-Preference-Language中设置语言为中文，可以让Lighthouse面板中的内容变为中文，Lighthouse导出的报告也会变成中文——来自译者的生活经验）
安装: Chrome
运行:打开Chrome 开发者工具，选择Lighthouse面板，点击“Analyze page load”（原文面板）

使用Chrome扩展程序

Chrome在Lighthouse集成到开发者工具之前就提供了Lighthouse的插件，并且有相似的功能
安装:从Chrome应用商店安装插件
运行:根据Lighthouse文档（后续也会加上这一部分的翻译）

使用Node CLI

Node CLI对Lighthouse运行时在配置和报告方面提供了最大的灵活性。如果想要更高级的用法，或者想要以自动化的方式运行Lighthouse，开发团队推荐用户使用Node CLI + Lighthouse
Lighthouse 依赖Node 16 LTS (16.x)或者更晚的版本
安装:

运行: lighthouse https://airhorner.com/
默认情况下，Lighthouse的报告是HTML文件。用户可以通过传参来控制报告的格式。
（原文中提供了大量的lighthouse参数，可以参见原文）。这里只提供一个简单的例子lighthouse <url> --output=json --output-path=./report.json表明了报告的格式和所在的位置（又是来自译者的生活经验）
错误报告说明
第一次运行 CLI 时，系统会询问 Lighthouse 是否可以匿名报告运行时发生的异常，选择退出不会影响Lighthouse的使用。
